🐭 囤囤鼠来哩 (TunTunMouse)
> 因为突发奇想，和小克一起做了这个项目，目的是为了提醒自己每天做游戏日常（笑死），然后基于朋友们的一起测试使用，功能也是越来越多了。
> 以下是哈基米生成的功能预设，我懒得自己打字了，感谢ai。
这是一名为二次元游戏玩家（囤囤鼠）量身定制的轻量化、Q版风格的资源管理与日常记录工具。无需复杂的操作，让你在指尖轻松掌握全服账号的资产动态，记录每一次抽卡的欢笑与“泪水”。
✨ 为什么选择“囤囤鼠”？
我们的设计理念是：萌即是正义，效率即是生命。
 * Q版治愈系 UI：采用圆润的卡片设计、活泼的 HSL 动态主题色以及精致的微动画，让管理数据不再枯燥。
 * 多维度管理：从每日任务到资源波动，从心情日记到跨端协作，你需要的全都有。
 * 极度轻量：纯前端实现，丝滑如原生的侧滑交互体验。
🚀 核心功能一览
1. 📋 任务清单 (Tasks)
 * 每日与长期任务：支持设置每日重置的“每日任务”或一次性的“成就任务”。
 * 快捷状态切换：一键勾选，看着进度条填满，成就感满满。
2. 💎 资源中心 (Resources)
 * 多游戏/多账号支持：可以添加多个游戏项目及其下属的多个账号。
 * 实时数据追踪：可视化展示资源（如原石、星琼等）的增减变化，甚至支持查看资源波动的趋势图表。
 * 日报分享：自动生成精美的“资源日报”卡片，一键分享你的囤货进度。
3. 📅 全能日历 (Calendar)
 * 历史回顾：通过日历视图快速查看过往的资源变动和日志记录。
 * 打卡标记：有记录的日子会自动打上小点，你的努力一目了然。
4. 📖 心情日记 (Diary)
 * 多类型分类：支持攻略、心得、吐槽、日常等多种标签分类。
 * 图文并茂：支持上传多张图片，记录欧气爆发的瞬间。
5. 🤝 实时协作 (Collaboration)
 * 房间共享：支持通过房间码进行多人协作，非常适合代练、共号或好友间互相监督。
🎨 个性化定制
我们尊重你的审美！在设置面板中，你可以：
 * 一键切换深色模式：保护视力，深夜肝图更舒适。
 * 自定义主题色：预设多种萌系配色，支持 HSL 动态微调。
 * 调节卡片透明度：打造属于你的磨砂玻璃质感。
🛠️ 技术细节
 * 原生交互：采用从右侧滑入的 detail-page 设计，完美模拟原生 App 的操作手感。
 * 本地存储：优先保证响应速度，支持导出备份。
 * 响应式布局：完美适配移动端（最佳体验宽度：480px 以内）。
🐭 结语
